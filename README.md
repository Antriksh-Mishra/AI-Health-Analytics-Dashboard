# 🏥 AI Health Analytics Dashboard

## 📌 Overview

AI Health Analytics Dashboard is a full-stack AI application that analyzes medical reports using OCR, Machine Learning, and Large Language Models.

Users can upload medical reports, visualize health metrics, understand lab values through AI-generated explanations, and monitor trends over time.

---

## 🚀 Features

- Medical Report Upload
- OCR-based Text Extraction
- Lab Value Detection
- Interactive Dashboard
- AI Report Explanation
- Health Trend Analysis
- Machine Learning Prediction
- PDF Report Generation
